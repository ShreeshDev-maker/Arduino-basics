# Arduino-basics
This will be a repo with simple arduino projects.
Beginners looking to start their electronics journey can easily reference this repo. 
Every project will have numerous comments and explanations

Lets start this amazing journey together!

Start with the basics then gradually move to advanced programs including understanding I2C, SPI, reading registers, etc

This repo is entirely managed and updated by me.
I will try my best to fully introduce beginners to advanced arduino coding from the ground-up.
This repo was created by me to fix the issue with other online video courses which only explain the basics and don't take you to an advanced level.
This repo is designed to be completely beginner- friendly.
The only prerequisites are-
What is arduino?   
Setting up the Arduino IDE   
Breadboarding and series and parallel circuits
